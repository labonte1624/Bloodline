# UI

This document describes the app user interface direction.

## Design Direction
- Keep the experience simple, focused, and powerful.
- Prioritize clarity, momentum, and visual discipline.
- Support mobile-first use while remaining strong on desktop.

## Visual Language
- Dark, sharp, and high-contrast aesthetic
- Bold typography for emphasis
- Minimal distractions and clean navigation

## Interaction Style
- Action-oriented screens
- Clear progression indicators
- Immediate feedback after workout completion
